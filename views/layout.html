Overview

Welcome to the Java Card online Simulator.

By Adam Noakes

Refactored by Weichao Gong

This section provides a basic documentation for the simulator.
Sections highlighted in red have not yet been documented.

Interact with the simulator by typing commands detailed in this section, and pressing enter.

Card Management

Command Function Link
cards Prints out the list of virtual smart cards that are available to the user.
newcard <name> Creates a new virtual smart card with the specified <name>.
deletecard <name> Deletes the virutal smart card with the specified <name>.
loadcard <name> Loads the virutal smart card with the specified <name>.
